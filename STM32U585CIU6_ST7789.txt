Dicas:
USER CODE BEGIN PV (variáveis globais)
USER CODE BEGIN PFP (Protótipo de funções)
USER CODE BEGIN 0 (Código auxiliar antes do main)
USER CODE BEGIN 1 (Antes do HAL_Init())
USER CODE BEGIN Init (Antes do Clock)
USER CODE BEGIN SysInit (Depois do Clock)
USER CODE BEGIN 2 (Aqui inicializa TUDO) O que a gente utiliza mais
USER CODE BEGIN WHILE (Loop principal)
USER CODE BEGIN 3 (Parte final do loop)
USER CODE BEGIN SPI1_Init 0 / 1 / 2 (Antes de configurar SPI)
USER CODE BEGIN MX_GPIO_Init_1 (Antes de configurar os pinos)
USER CODE BEGIN MX_GPIO_Init_2 (Depois de configurar GPIO)
USER CODE BEGIN 4 (OUTRO MAIS IMPORTANTE) Aqui as funções que criamos
USER CODE BEGIN Error_Handler_Debug (Só para debug)
USER CODE BEGIN 6 (Para assert (debug avançado))
